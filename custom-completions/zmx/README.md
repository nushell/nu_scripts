# zmx completions

Completions for [zmx](https://github.com/neurosnap/zmx). `zmx` is a terminal
session manager that lets you attach and detach from persistent shell sessions
without killing them.

These completions include every `zmx` subcommand (and its short alias),
their flags, and dynamic completion of live session names via
`zmx list --short`.

## Usage

```nushell
use zmx-completions.nu *
```

or

```nushell
source zmx-completions.nu
```
